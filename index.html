How the course ready helping you prepare for a career in IT

In the past of my life, I had never learnt about computer science or programming. Therefore, all the IT knowledge that lecture gives are really new challenges to me. However, teacher help me a lot and lets me lean it much easy: Without a stable and tough base, it will be very difficult to learn further more knowledge, I may be confused in some concepts.

In the major module of the course, introduction to computing give us the fundamental sense of programming. During lessons, we learn how to use python, one of the program languages which is friendly for the beginners, just like me.

To be honest, It is a quite difficult task for me to learn a programming language with limited computer skills. However, the professor guided us steps by steps in each tutorial from basics to advance. The experience to master the skills of coding can guide me in the further career because programming is an essential skill in future day, just like Microsoft Office.

Consideration of how your current capabilities match your aspirations

First of all, I want to consider that how my current capabilities match my aspirations. Before I choose this course: I didn't learn any programming in my previous school life. This is my first time to learn computer science so I would not give myself a high target to achieve with. Now my situation is learning the basic programming of python, like loop, define, int/float. It is clear that I can practice more or self-study via tutorial video after lessons. For instance, some opening file method, Internet data handling, structure markup processing tools, http module, etc. My current capabilities may not completely match my aspirations, since I am initially touch with programming.

Recognition that where there is a shortfall in current capabilities that needs addressing

It is my first time to learn programming, and I haven't prepare it before so I usually confuse during the lesson. This such an abstract concept for me and may take a long time to digest. Thus, it quite important for self-study after the lessons.

Nevertheless, I think I still have improvement in this part. After some repeat studying of the tutorials, I started to alter the original programs and run it without any bugs. This give me the sense of success and encouragement for me to push forward in another program. Therefore, I want to improve my debugging skill to make more possibility of programming.

Besides, I have a basic concept of what taught in lecture but I was not familiarized with some practical skills such as HTML and CSS. This is caused by lack of exercise in practical part. And I am willing to practice harder by visit some useful websites, like Code Academy in the coming days.
